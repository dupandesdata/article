Dokumentasi Aplikasi OCR dengan Pemotongan Opsional

Deskripsi Aplikasi

Aplikasi ini adalah alat berbasis web yang memungkinkan pengguna untuk mengunggah gambar, memotong gambar tersebut jika diperlukan, dan kemudian menggunakan teknologi Optical Character Recognition (OCR) untuk mengonversi teks dalam gambar menjadi teks yang dapat diedit. Aplikasi ini dibangun menggunakan HTML, CSS, dan JavaScript dengan framework Vue.js dan Tesseract.js.

Fitur Utama

  1. Unggah Gambar: Pengguna dapat mengunggah gambar dari perangkat mereka.
  2. Pemotongan Gambar: Pengguna dapat memotong gambar sebelum melakukan konversi teks.
  3. Konversi Teks: Menggunakan Tesseract.js untuk mengonversi gambar menjadi teks.
  4. Antarmuka Responsif: Desain yang responsif dan ramah pengguna.

Struktur HTML

Aplikasi ini menggunakan struktur HTML yang sederhana dengan elemen-elemen berikut:

  • <head>: Berisi metadata, judul, dan link ke stylesheet serta skrip yang diperlukan.
  • <body>: Berisi elemen-elemen antarmuka pengguna seperti judul, input file, gambar, tombol, dan area untuk menampilkan teks yang telah dikonversi.

Penjelasan JavaScript

Aplikasi ini menggunakan Vue.js untuk mengelola state dan interaksi pengguna. Berikut adalah penjelasan detail tentang bagian JavaScript:

  • el: Menentukan elemen DOM yang akan dikelola oleh Vue.
  • data: Menyimpan state aplikasi, termasuk URL gambar, URL gambar yang dipotong, teks yang telah dikonversi, dan status pemrosesan.

Metode

  1. handleImageUpload(event)

    • Mengelola unggahan gambar.
    • Membaca file gambar dan mengubahnya menjadi URL yang dapat digunakan untuk ditampilkan.
    • Menginisialisasi pemotongan jika ada gambar baru yang diunggah.

  2. initCropper()

    • Menginisialisasi Cropper.js untuk memungkinkan pemotongan gambar.
    • Mengatur isCropping menjadi true untuk menampilkan kontrol pemotongan.

  3. cropImage()

    • Mengambil gambar yang telah dipotong dan mengubahnya menjadi URL.
    • Menghentikan mode pemotongan dan menghancurkan instance Cropper.

  4. cancelCrop()

    • Menghentikan pemotongan dan menghapus instance Cropper jika ada.
    • Mengatur isCropping dan croppedImageUrl ke nilai awal.

  5. convertToText()

    • Mengonversi gambar (baik yang dipotong atau tidak) menjadi teks menggunakan Tesseract.js.
    • Menangani status konversi dan menampilkan hasilnya.
    • Menangani kesalahan yang mungkin terjadi selama proses konversi.

Penggunaan Tesseract.js

javascript
1Tesseract.recognize(imageToConvert)
  2  .then(({ data: { text } }) => {
  3    this.convertedText = text;
  4  })
  5  .catch(error => {
  6    console.error('Error during OCR:', error);
  7    this.convertedText = 'Error occurred during text conversion.';
  8  })
  9  .finally(() => {
  10    this.isConverting = false;
  11  });
  • Tesseract.recognize(): Memanggil fungsi untuk memulai proses OCR pada gambar yang diberikan.
  • then(): Menangani hasil konversi dan menyimpan teks yang dihasilkan.
  • catch(): Menangani kesalahan yang mungkin terjadi selama proses OCR.
  • finally(): Mengatur status konversi kembali ke false setelah proses selesai.

Kesimpulan

Aplikasi OCR ini memberikan cara yang mudah dan efisien untuk mengonversi teks dari gambar menjadi format yang dapat diedit. Dengan fitur pemotongan opsional, pengguna dapat memastikan bahwa hanya bagian gambar yang relevan yang akan diproses, meningkatkan akurasi konversi. Aplikasi ini menggunakan teknologi modern seperti Vue.js dan Tesseract.js untuk memberikan pengalaman pengguna yang responsif dan intuitif.
