Random Number Generator

Dokumentasi ini menjelaskan cara kerja aplikasi Random Number Generator yang dibangun menggunakan Vue.js dan Tailwind CSS. Aplikasi ini memungkinkan pengguna untuk menghasilkan angka acak dalam rentang yang ditentukan, dengan opsi untuk menghasilkan angka unik atau duplikat.


Fungsi Utama Aplikasi

  • Data

    • min: Nilai minimum untuk rentang angka.
    • max: Nilai maksimum untuk rentang angka.
    • mode: Mode penghasilan angka (random atau unique).
    • result: Menyimpan hasil angka yang dihasilkan.
    • usedNumbers: Set untuk menyimpan angka yang sudah digunakan (hanya untuk mode unique).

  • Computed Properties

    • remainingNumbers: Menghitung jumlah angka unik yang tersisa.
    • isButtonDisabled: Menentukan apakah tombol "Generate Number" dinonaktifkan.

  • Watchers

    • Memantau perubahan pada mode, min, dan max untuk mengosongkan usedNumbers.

  • Methods

    • generate(): Menghasilkan angka acak berdasarkan mode yang dipilih.
      • Jika mode adalah unique dan semua angka telah digunakan, tampilkan alert.
      • Menghasilkan angka acak dan memeriksa apakah angka tersebut sudah digunakan (jika mode unique).

Tampilan Antarmuka

  • Input untuk Rentang Angka

    • Dua input untuk nilai minimum dan maksimum.
    • Label yang jelas untuk setiap input.

  • Mode Pilihan

    • Dropdown untuk memilih mode (random atau unique).

  • Tombol Generate

    • Tombol untuk menghasilkan angka dengan efek hover dan transisi.

  • Hasil

    • Menampilkan angka yang dihasilkan dengan animasi fade-in.
    • Menampilkan jumlah angka unik yang tersisa jika mode unique dipilih.
