Dokumentasi Aplikasi Excel Converter

Aplikasi ini adalah alat berbasis web yang memungkinkan pengguna untuk mengonversi file Excel ke berbagai format seperti JSON, HTML Table, PDF, atau PNG. Aplikasi ini dibangun menggunakan Vue.js dan beberapa pustaka JavaScript untuk memproses file Excel dan menghasilkan output yang diinginkan.

Fitur Utama

  1. Unggah File Excel: Pengguna dapat mengunggah file Excel (.xlsx atau .xls).
  2. Format Output: Pengguna dapat memilih format output yang diinginkan (JSON, HTML, PDF, PNG).
  3. Ekspor Data: Mengonversi dan mengekspor data ke format yang dipilih.
  4. Tampilan Prabaca: Menampilkan prabaca data dalam format tabel sebelum diekspor.
  5. Header/Footer untuk PDF/PNG: Opsi untuk menambahkan header dan footer saat mengekspor ke PDF atau PNG.

Struktur HTML

Aplikasi ini menggunakan struktur HTML yang sederhana dengan elemen-elemen berikut:

  • <head>: Berisi metadata, judul, dan link ke stylesheet serta skrip yang diperlukan.
  • <body>: Berisi elemen-elemen antarmuka pengguna seperti input file, tombol, dan area untuk menampilkan prabaca data.

Penjelasan JavaScript

Aplikasi ini menggunakan Vue.js untuk mengelola state dan interaksi pengguna. Berikut adalah penjelasan detail tentang bagian JavaScript:

exportData()

    • Menangani ekspor data ke format yang dipilih (JSON, HTML, PDF, PNG).
    • Menggunakan Blob untuk membuat file yang dapat diunduh untuk format JSON dan HTML.
    • Menggunakan html2pdf untuk mengekspor ke PDF.
    • Menggunakan html2canvas untuk mengekspor ke PNG.

Fungsi getExportCode()

Fungsi ini menghasilkan kode ekspor berdasarkan format yang dipilih (JSON atau HTML). Berikut adalah penjelasan yang lebih baik dan perbaikan pada penulisan kode:

javascript
1const getExportCode = () => {
    2    if (selectedFormat.value === 'JSON') {
    3        const jsonData = tableData.value.map(row => {
    4            const obj = {};
    5            tableHeaders.value.forEach((header, index) => {
    6                obj[header] = row[index];
    7            });
    8            return obj;
    9        });
    10        return `const data = ${JSON.stringify(jsonData, null, 2)};`;
    11    } else if (selectedFormat.value === 'HTML') {
    12        return `<table>
    13  <thead>
    14    <tr>
    15      ${tableHeaders.value.map(header => `<th>${header}</th>`).join('')}
    16    </tr>
    17  </thead>
    18  <tbody>
    19    ${tableData.value.map(row => `
    20    <tr>
    21      ${row.map(cell => `<td>${cell}</td>`).join('')}
    22    </tr>`).join('')}
    23  </tbody>
    24</table>`;
    25    }
    26    return '';
    27}
  • JSON: Mengonversi data tabel menjadi objek JSON dan mengembalikannya dalam format string.
  • HTML: Mengonversi data tabel menjadi string HTML yang valid, termasuk elemen <table>, <thead>, dan <tbody>.

Aplikasi Excel Converter ini memberikan cara yang mudah dan efisien untuk mengonversi file Excel ke berbagai format seperti JSON, HTML, PDF, dan PNG.
