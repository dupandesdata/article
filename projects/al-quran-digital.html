Berikut adalah dokumentasi untuk aplikasi Al-Qur'an Digital yang dibangun menggunakan Vue.js dan Vue Router. Aplikasi ini memungkinkan pengguna untuk mencari dan membaca surah dari Al-Qur'an.

Struktur HTML

  • Header
    • Menampilkan judul aplikasi "Al-Qur'an Digital".
  • Main
    • Menampilkan komponen yang sesuai dengan rute yang dipilih.
  • Footer
    • Menampilkan informasi hak cipta.

Komponen Vue

  1. Home Component

    • Template: Menampilkan input pencarian dan daftar surah.
    • Fungsi Utama:
      • Mengambil data surah dari API.
      • Menyaring surah berdasarkan input pencarian.
      • Menampilkan popup untuk bacaan terakhir jika ada.

  2. Surah Detail Component

    • Template: Menampilkan detail surah dan ayat-ayatnya.
    • Fungsi Utama:
      • Mengambil detail surah berdasarkan ID.
      • Menampilkan ayat dengan teks Arab, Latin, dan terjemahan.
      • Menyediakan navigasi untuk surah sebelumnya dan selanjutnya.

Penggunaan Vue Router

  • Rute:
    • /: Menampilkan komponen Home.
    • /surah/:id: Menampilkan detail surah berdasarkan ID.

Fungsi Penting dalam Script

  • fetchSurahs: Mengambil data surah dari API.
  • filteredSurahs: Menghitung surah yang sesuai dengan query pencarian.
  • playAudio: Memutar audio dari ayat yang dipilih.
  • updateLocalStorage: Menyimpan surah dan ayat terakhir yang dibaca ke local storage.

Gaya CSS

  • Menggunakan Tailwind CSS untuk styling.
  • Menyediakan gaya responsif untuk tampilan yang lebih baik di berbagai ukuran layar.

Kesalahan yang Ditemui

  • 404 Client Error: Terdapat kesalahan saat memuat ikon dari URL yang tidak ditemukan. Pastikan URL ikon yang digunakan valid.

Kesimpulan

Aplikasi Al-Qur'an Digital ini memberikan pengalaman membaca Al-Qur'an yang interaktif dan mudah diakses. Dengan fitur pencarian dan detail surah, pengguna dapat dengan mudah menemukan dan membaca ayat-ayat yang mereka inginkan.
