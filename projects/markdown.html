Dokumentasi Aplikasi Markdown to HTML

Deskripsi Aplikasi

Aplikasi ini adalah contoh penggunaan Vue.js untuk mengambil data dari API dan menampilkan konten dalam format Markdown atau HTML. Pengguna dapat memasukkan URL API, memilih jenis output, dan melihat hasilnya di antarmuka yang responsif.

Fitur Utama

  1. Input URL API: Pengguna dapat memasukkan URL dari mana data akan diambil.
  2. Pilihan Jenis Output: Pengguna dapat memilih untuk menampilkan data sebagai teks biasa atau HTML.
  3. Pengambilan Data: Menggunakan Axios untuk melakukan permintaan HTTP ke API.
  4. Salin ke Clipboard: Pengguna dapat menyalin konten yang ditampilkan ke clipboard.
  5. Menampilkan Konten: Menampilkan konten yang diambil dalam format yang sesuai.

Struktur HTML

Aplikasi ini menggunakan struktur HTML yang sederhana dengan elemen-elemen berikut:

  • <head>: Berisi metadata, judul, dan link ke stylesheet serta skrip yang diperlukan.
  • <body>: Berisi elemen-elemen antarmuka pengguna seperti input, tombol, dan area untuk menampilkan konten.

Metode

  1. fetchData()

    • Mengambil data dari URL API yang dimasukkan.
    • Mengatur status loading dan menangani kesalahan jika terjadi.
    • Mengonversi data Markdown menjadi HTML jika diperlukan.
    javascript
    1fetchData() {
  2  this.error = null;
  3  this.loading = true;
  4
  5  axios.get(this.apiUrl)
  6    .then(response => {
  7      this.outputContent = this.outputType === 'html'
  8        ? this.markdownToHtml(response.data, false)
  9        : this.markdownToHtml(response.data, true);
  10    })
  11    .catch(err => {
  12      this.error = 'Terjadi kesalahan: ' + err.message;
  13    })
  14    .finally(() => {
  15      this.loading = false;
  16    });
  17}

  2. copyToClipboard()

    • Menyalin konten yang ditampilkan ke clipboard.
    • Menggunakan elemen textarea sementara untuk menyalin teks.
    javascript
    1copyToClipboard() {
  2  const tempTextArea = document.createElement('textarea');
  3  tempTextArea.value = this.outputType === 'html'
  4    ? this.outputContent
  5    : this.outputContent.replace(/</g, '&lt;').replace(/>/g, '&gt;'); // Encode HTML tags
  6  document.body.appendChild(tempTextArea);
  7  tempTextArea.select();
  8  document.execCommand('copy');
  9  document.body.removeChild(tempTextArea);
  10  alert('Teks berhasil disalin ke clipboard!');
  11}

  3. markdownToHtml(markdown, isText)

    • Mengonversi teks Markdown menjadi HTML.
    • Menggunakan regex untuk mengganti sintaks Markdown dengan elemen HTML yang sesuai.
    javascript
    1markdownToHtml(markdown, isText) {
  2  markdown = markdown.replace(/###### (.+)/g, isText ? '###### $1' : '<h6 class="text-base font-normal text-gray-300">$1</h6>');
  3  // ... (proses penggantian lainnya)
  4  return markdown;
  5}

Kesimpulan

Aplikasi ini memberikan cara yang mudah untuk mengambil dan menampilkan data dari API dalam format Markdown atau HTML. Dengan menggunakan Vue.js dan Axios, aplikasi ini menawarkan antarmuka yang responsif dan interaktif, memungkinkan pengguna untuk berinteraksi dengan data secara efisien.
