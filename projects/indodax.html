Dokumentasi Fungsi API Indodax dalam Google Apps Script

Deskripsi Fungsi

Fungsi ini digunakan untuk berinteraksi dengan API Indodax, sebuah platform perdagangan cryptocurrency. Fungsi ini memungkinkan pengguna untuk melakukan berbagai tindakan seperti mendapatkan saldo, melakukan order, membatalkan order, dan mengambil riwayat perdagangan.

Parameter Fungsi

Fungsi ini menerima beberapa parameter yang menentukan tindakan yang akan dilakukan:

  • action: Tindakan yang ingin dilakukan (misalnya, "getBalance", "order", "cancel", dll.).
  • type: Jenis order atau pasangan mata uang (misalnya, "buy", "sell", atau pasangan mata uang seperti "btc_idr").
  • pair: Pasangan mata uang yang akan diperdagangkan (misalnya, "btc_idr").
  • amount: Jumlah yang akan diperdagangkan.
  • price: Harga untuk order (diperlukan untuk order limit).
  • market: Boolean yang menunjukkan apakah order adalah market order atau limit order.

Cara Kerja

Fungsi ini bekerja dengan cara berikut:

  1. Inisialisasi: Mengambil API key dan secret key dari data pengguna.
  2. Membangun Request: Bergantung pada nilai parameter action, fungsi ini membangun objek requestBody yang berisi informasi yang diperlukan untuk permintaan API.
  3. Mengirim Permintaan: Fungsi request() digunakan untuk mengirim permintaan ke API Indodax dengan menggunakan metode POST.
  4. Menangani Respon: Mengambil dan mengembalikan data dari API berdasarkan tindakan yang diminta.

Fungsi Pendukung

  1. request()

    • Mengirim permintaan ke API Indodax.
    javascript
    1function request() {
  2  const payloadString = toQueryString(requestBody);
  3  const signature = hmac_sha512(payloadString);
  4  const headers = { Key: apiKey, Sign: signature };
  5  const options = {
  6    method: "post",
  7    headers: headers,
  8    payload: payloadString
  9  };
  10
  11  const response = UrlFetchApp.fetch(url, options);
  12  const responseData = JSON.parse(response.getContentText());
  13  return responseData;
  14}

  2. hmac_sha512(payloadString)

    • Menghasilkan tanda tangan HMAC SHA-512 untuk keamanan.
    javascript
    1function hmac_sha512(payloadString) {
  2  var hmacBytes = Utilities.computeHmacSignature(
  3    Utilities.MacAlgorithm.HMAC_SHA_512,
  4    payloadString,
  5    secretKey
  6  );
  7  var hmacString = "";
  8  for (var i = 0; i < hmacBytes.length; i++) {
  9    hmacString += ("0" + (hmacBytes[i] & 0xff).toString(16)).slice(-2);
  10  }
  11  return hmacString;
  12}

  3. toQueryString(obj)

    • Mengonversi objek menjadi string query untuk permintaan API.
    javascript
    1function toQueryString(obj) {
  2  return Object.keys(obj)
  3    .map(
  4      (key) => encodeURIComponent(key) + "=" + encodeURIComponent(obj[key])
  5    )
  6    .join("&");
  7}

Kesimpulan

Fungsi indodax adalah alat yang kuat untuk berinteraksi dengan API Indodax, memungkinkan pengguna untuk melakukan berbagai tindakan terkait perdagangan cryptocurrency. Dengan menggunakan metode yang aman dan terstruktur, fungsi ini memberikan cara yang efisien untuk mengelola perdagangan dan mendapatkan informasi saldo.
